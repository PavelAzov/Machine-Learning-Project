# Predictive Modeling: Прогнозирование рыночной стоимости

## 📌 Описание проекта
Данный проект посвящен разработке модели машинного обучения для прогнозирования стоимости объектов на основе набора технических параметров. В ходе работы был реализован полный цикл разработки: от глубокого аудита качества данных до финальной оценки бизнес-метрик.

**Ключевая цель:** Построить стабильную модель, способную предсказывать цену с высокой точностью, минимизируя влияние аномалий и шума в данных.

## 🛠 Технологический стек
*   **Язык:** Python 3.x
*   **Библиотеки:** Pandas, NumPy, Scikit-learn, Seaborn, Matplotlib
*   **Алгоритм:** Random Forest Regressor (ансамблевое обучение)
*   **Методология:** EDA, Statistical Outlier Detection (IQR), Feature Engineering

## 📊 Этапы реализации
1.  **Технический аудит данных:** Выявление дубликатов, пропущенных значений и первичный анализ распределений.
2.  **Двухэтапная очистка:**
    *   Удаление технического мусора (дубликаты и NaN).
    *   Статистическая фильтрация экстремальных выбросов методом межквартильного размаха (IQR).
3.  **Разведочный анализ (EDA):** Анализ корреляций между признаками `para1-para4` и целевой переменной `price`.
4.  **Моделирование:** Обучение модели Random Forest с подбором параметров для предотвращения переобучения.
5.  **Оценка результатов:** Анализ ошибок модели (RMSE) и коэффициента детерминации (R^2).

## 📈 Результаты и метрики
После проведения глубокой очистки данных и настройки модели были достигнуты следующие показатели:
*   **R² (Коэффициент детерминации):** `0.6997` (~70% вариативности цены объясняется моделью).
*   **RMSE (Root Mean Squared Error):** `117.03` (Средняя квадратичная ошибка в единицах цены).

> **Аналитический вывод:** Модель демонстрирует высокую предсказательную силу. Основной вклад в точность внесла предварительная обработка данных и удаление аномальных значений, что позволило повысить устойчивость прогноза.
